# Site do Psicólogo André Lemos

Site clone do psicólogo André Lemos - www.psicologobrasiliadf.com

## Estrutura do Projeto

```
andre-psico/
├── index.html              # Página inicial
├── sobre-mim.html           # Sobre o psicólogo
├── servicos.html            # Serviços oferecidos
├── artigos.html             # Artigos e blog
├── agendamento.html         # Sistema de agendamento
├── avaliacoes.html          # Avaliações de pacientes
├── testes.html              # Testes psicológicos de rastreio
├── instagram.html           # Página do Instagram
├── css/
│   └── style.css            # Estilos principais
├── js/
│   └── script.js            # Funcionalidades JavaScript
├── images/                  # Diretório de imagens
└── README.md                # Este arquivo
```

## Funcionalidades Implementadas

### ✅ Páginas Criadas
- **Página Inicial**: Hero section, serviços, artigos em destaque, contato
- **Sobre Mim**: Biografia profissional, formações, áreas de atuação
- **Serviços**: Lista completa de serviços com filtros interativos
- **Artigos**: Blog com categorias e sistema de inscrição
- **Agendamento**: Sistema multi-step para agendamento de consultas
- **Avaliações**: Depoimentos de pacientes com sistema de avaliação
- **Testes**: Testes psicológicos de rastreio interativos
- **Instagram**: Feed simulado e informações de contato

### ✅ Funcionalidades Técnicas
- **Design Responsivo**: Adaptado para desktop, tablet e mobile
- **Bootstrap 5**: Framework CSS moderno
- **JavaScript Interativo**: Animações, filtros, formulários
- **SEO Otimizado**: Meta tags, semântica HTML5
- **Acessibilidade**: Estrutura acessível e navegação por teclado

### ✅ Conteúdo Clonado
- Todas as informações do site original
- Textos idênticos sobre serviços e tratamentos
- Informações de contato e endereço
- Estrutura de navegação completa

## Tecnologias Utilizadas

- **HTML5**: Semântica e estrutura
- **CSS3**: Estilos modernos com gradientes e animações
- **Bootstrap 5**: Grid system e componentes
- **Font Awesome**: Ícones
- **JavaScript Vanilla**: Interatividade
- **Google Fonts**: Tipografia

## Como Usar

1. **Local**: Abra `index.html` diretamente no navegador
2. **Servidor Local**: Use XAMPP, WAMP ou similar para rodar em servidor local
3. **Produção**: Faça upload para seu servidor web
